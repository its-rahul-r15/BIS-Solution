# BIS Solutions Website - Setup Instructions

## Dependencies to Install

Please run these commands in your terminal one by one:

### 1. Install Tailwind CSS
```bash
npm install -D tailwindcss postcss autoprefixer
npx tailwindcss init -p
```

### 2. Install React Icons
```bash
npm install react-icons
```

## After Installation

Once the dependencies are installed, I will:
1. Configure Tailwind CSS
2. Create all React components
3. Build the complete website using your images
